Сумма факториалов

Дано натуральное число $n$. Напишите программу, которая выводит значение суммы $1!+2!+3!+\ldots+n!$.

Формат входных данных
На вход программе подается одно натуральное число.

Формат выходных данных
Программа должна вывести значение суммы $1!+2!+3!+\ldots+n!$.

Примечание 1. Факториалом натурального числа $n$, называется произведение всех натуральных чисел от \( 1 \) до \(n\), то есть \[n!=1\cdot2\cdot3\cdot…\cdot n\]

Примечание 2. Задачу можно решить без вложенного цикла. Напишите две версии программы =)


Sample Input:
3

Sample Output:
9



Memory limit: 256 MB
Time limit: 15 seconds
