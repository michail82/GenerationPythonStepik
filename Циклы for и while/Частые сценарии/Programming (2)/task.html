Асимптотическое приближение

На вход программе подается натуральное число $n$. Напишите программу, которая вычисляет значение выражения\[\left(1+\dfrac12 + \dfrac13 + \ldots + \dfrac{1}{n} \right) - \ln (n).\]

Примечание. Для вычисления натурального логарифма воспользуйтесь функцией log(n), которая находится в модуле math.


Sample Input:
10

Sample Output:
0.6263831609742079


Sample Input:
1

Sample Output:
1.0


Sample Input:
100

Sample Output:
0.5822073316515288
