Факториал

На вход программе подается натуральное число $n$. Напишите программу, которая вычисляет $n!$.

Входные данные
На вход программе подается натуральное число $n, \, (n \le 12)$.

Выходные данные
Программа должна вывести единственное число в соответствии с условием задачи.

Примечание. Факториалом натурального числа $n$, называется произведение всех натуральных чисел от \( 1 \) до \(n\), то есть \[n!=1\cdot2\cdot3\cdot…\cdot n\]


Sample Input:
3

Sample Output:
6


Sample Input:
1

Sample Output:
1


Sample Input:
2

Sample Output:
2



Memory limit: 256 MB
Time limit: 15 seconds
