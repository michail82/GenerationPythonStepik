Знакочередующаяся сумма

На вход программе подается натуральное число $n$. Напишите программу вычисления знакочередующей суммы \[1-2+3-4+5-6 + \ldots + (-1)^{n+1}n.\]

Входные данные
На вход программе подается натуральное число $n$.

Выходные данные
Программа должна вывести единственное число в соответствии с условием задачи.


Sample Input:
1

Sample Output:
1


Sample Input:
5

Sample Output:
3


Sample Input:
3

Sample Output:
2



Memory limit: 256 MB
Time limit: 15 seconds
