Звездный прямоугольник

На вход программе подается натуральное число $n$.

Напишите программу, которая печатает звездный прямоугольник размерами $n \times 19$.

Формат входных данных
На вход программе подаётся натуральное число $n \in [1; \, 20]$ — высота звездного прямоугольника.

Формат выходных данных
Программа должна вывести звездный прямоугольник размерами $n \times 19$.

Подсказка. Для печати звездной линии используйте умножение строки на число.

1

Sample Output:

2

*******************

3

*******************
*******************
