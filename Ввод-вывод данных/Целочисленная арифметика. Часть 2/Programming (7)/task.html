Перестановка цифр

Дано трехзначное число $\overline{abc}$, в котором все цифры различны. Напишите программу, которая выводит шесть чисел, образованных при перестановке цифр заданного числа.

Формат входных данных
На вход программе подаётся положительное трёхзначное целое число, все цифры которого различны.

Формат выходных данных
Программа должна вывести шесть чисел, образованных при перестановке цифр заданного числа в следующем порядке: $abc, \, acb, \, bac, \, bca, \, cab, \, cba$.


Sample Input:
123

Sample Output:
123
132
213
231
312
321


Sample Input:
987

Sample Output:
987
978
897
879
798
789



Memory limit: 256 MB
Time limit: 15 seconds
