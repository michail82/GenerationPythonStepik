Список четных

На вход программе подается четное число $n, \, n \ge 2$. Напишите программу, которая выводит список четных чисел

 [2, 4, 6, ..., n].

Формат входных данных
На вход программе подается четное натуральное число.

Формат выходных данных
Программа должна вывести текст в соответствии с условием задачи.


Sample Input:
2

Sample Output:
[2]


Sample Input:
4

Sample Output:
[2, 4]


Sample Input:
8

Sample Output:
[2, 4, 6, 8]



Memory limit: 256 MB
Time limit: 15 seconds
