Звездная рамка

На вход программе подается натуральное число $n$. Напишите программу, которая печатает звездную рамку размерами $n \times 19$.

Формат входных данных
На вход программе подаётся натуральное число $n \in [3; \, 19]$ — высота звездной рамки.

Формат выходных данных
Программа должна вывести звездную рамку размерами $n \times 19$.

Подсказка. Для печати зведной линии используйте умножение строки на число.


Sample Input:
3

Sample Output:
*******************
*                 *
*******************


Sample Input:
4

Sample Output:
*******************
*                 *
*                 *
*******************


Sample Input:
5

Sample Output:
*******************
*                 *
*                 *
*                 *
*******************



Memory limit: 256 MB
Time limit: 15 seconds
