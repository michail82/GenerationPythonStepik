Значение функции

На вход программе подается натуральное число $n$, а затем $n$ целых чисел. Напишите программу, которая для каждого из введенного числа $x$ выводит значение функции $f(x) = x^2 + 2x + 1$, каждое на отдельной строке.

Формат входных данных
На вход программе подаются натуральное число $n$, а затем $n$ целых чисел, каждое на отдельной строке.

Формат выходных данных
Программа должна вывести сначала введенные числа, затем пустую строку, а затем соответствующие значения функции.

Примечание. Для первого теста имеем: $$f(1) = 1^2 + 2\cdot 1 + 1 = 4, \, f(2) = 2^2 + 2\cdot 2 + 1 = 9, \, f(3) = 3^2 + 2 \cdot 3 + 1 = 16, \ldots $$


Sample Input:
5
1
2
3
4
5

Sample Output:
1
2
3
4
5

4
9
16
25
36



Memory limit: 256 MB
Time limit: 15 seconds
