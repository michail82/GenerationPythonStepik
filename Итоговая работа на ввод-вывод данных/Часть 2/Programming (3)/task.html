Размножение n-ок

Напишите программу, которая считывает целое положительное число $n, \, n \in [1; \, 9]$ и выводит значение числа $n+\overline{nn}+\overline{nnn}$.

Формат входных данных
На вход программе подаётся одно целое положительное число $n, \, n \in [1; \, 9]$.

Формат выходных данных
Программа должна вывести число $n+\overline{nn}+\overline{nnn}$.

Примечание. Для первого теста $1 + 11 + 111 = 123$.


Sample Input:
1

Sample Output:
123


Sample Input:
2

Sample Output:
246


Sample Input:
3

Sample Output:
369



Memory limit: 256 MB
Time limit: 15 seconds
