451 градус по Фаренгейту 

У известного американского писателя Рэя Бредбери есть роман «451 градус по Фарангейту». Напишите программу, которая определяет, какой температуре по шкале Цельсия соответствует указанное значение по шкале Фаренгейта.

Используйте формулу для перевода: $C = \dfrac{5}{9}\left(F - 32\right)$.

Формат входных данных
На вход программе подаётся вещественное число градусов по шкале Фаренгейта.

Формат выходных данных
Программа должна вывести число градусов по шкале Цельсия.


Sample Input:
450.55

Sample Output:
232.5277777777778


Sample Input:
113.75

Sample Output:
45.41666666666667


Sample Input:
275.0

Sample Output:
135.0
